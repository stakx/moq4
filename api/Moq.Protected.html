Namespace Moq.Protected

Classes

ItExpr

Allows the specification of a matching condition for an argument in a protected member setup, rather than a specific argument value. "ItExpr" refers to the argument being matched.

ItExpr.Ref<TValue>

Contains matchers for langword_csharp_ref (C#) / langword_csharp_ByRef (VB.NET) parameters of type TValue.

ProtectedExtension

Enables the Protected() method on Mock<T>, allowing setups to be set for protected members by using their name as a string, rather than strong-typing them which is not possible due to their visibility.

Interfaces

IProtectedAsMock<T, TAnalog>

Allows setups to be specified for protected members (methods and properties) seen through another type with corresponding members (that is, members having identical signatures as the ones to be set up).

IProtectedMock<TMock>

Allows setups to be specified for protected members by using their name as a string, rather than strong-typing them which is not possible due to their visibility.
